1.Task Monitor — macOS Build (v6-final-mac7)

Что нового
- Всплывающий комментарий-«облачко» с хвостиком: непрозрачный фон, адаптация к теме (светлая/тёмная), неограниченный размер.
- Узкая колонка «Задача» (в 2 раза меньше предыдущей), место перераспределено между DEV/DEMO/LT/PROD.
- История изменений в человекочитаемом виде.

Запуск
1) Распакуйте архив:
   unzip task-monitor-modern-v6-final-mac7.zip
   cd task-monitor-modern-v6-final-mac7

2) Создайте и активируйте venv:
   python -m venv .venv
   source .venv/bin/activate

3) Установите зависимости:
   pip install -r requirements.txt

4) Запустите сервер:
   python run.py
   Откройте http://127.0.0.1:8000

Примечания
- В seed-данных есть длинный комментарий для проверки всплывающего окна.
- CSV экспорт с BOM (корректно открывается в Excel).


v6-final-mac8 — фиксы
- Комментарии ответственных корректно сохраняются и подтягиваются при редактировании (HTML-экранирование + надёжная сериализация).
- Всплывающий комментарий открывается непосредственно рядом с фамилией (вертикальное центрирование, позиционирование справа).
- Журнал изменений обогащён: отражает добавления/удаления/изменения комментариев с указанием этапа и ответственного.


v6-final-mac9 — фиксы стабильности
- Назначения/комментарии не удаляются, если JavaScript сериализация не сработала (guard): бэкенд сохраняет предыдущие данные.
- Форма теперь перед отправкой выставляет assign_guard=1 и сериализует assign_json.
- Всплывающий комментарий позиционируется вплотную справа от фамилии, с учётом видимой области экрана.
